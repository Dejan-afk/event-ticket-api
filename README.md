# Event Ticket Management API

A backend-focused event and ticket management system built with Symfony.
The goal of this project is to demonstrate modern backend architecture patterns, API design, and asynchronous processing.

This repository is designed as a portfolio-quality backend project that reflects real-world application design and engineering practices.

## Project Goals

- events
- ticket bookings
- ticket lifecycle
- availability tracking
- asynchronous notifications

## Backend Engineering Goals

- API architecture
- domain modeling
- asynchronous processing
- performance optimization
- database querying
- security and access control
